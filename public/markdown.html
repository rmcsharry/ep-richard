Markdown cheat sheet

Bullets

Asterisks get turned into nice bullets.

So this:

  * Point one
  * Point two
  * Point three

Becomes this:

  • Point one
  • Point two
  • Point three

Numbered lists

Just write the raw numbers:

So this:

  1. Point one
  2. Point two
  3. Point three

Becomes this:

  1. Point one
  2. Point two
  3. Point three
